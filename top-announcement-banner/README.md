# Top Announcement Banner

A small WordPress plugin that displays a top announcement banner for campaigns, offers, and seasonal promotions.

## Features

- Show a fixed banner at the top of the site
- Custom announcement text
- Optional CTA button with URL
- Configurable background and text colors
- Dismissible banner with local storage persistence
- Simple admin settings page

## Installation

1. Copy the `top-announcement-banner` folder into your WordPress `wp-content/plugins` directory.
2. Activate the plugin from the WordPress admin Plugins screen.
3. Visit Settings > Announcement Banner to configure your message.

## Usage

- Enable or disable the banner from the settings page
- Add campaign text and button link for promotions
- Use the color fields to match your brand style
- Choose whether visitors can dismiss the banner

## Notes

- The banner appears on the frontend only when enabled.
- Dismissal is stored in local storage, so returning visitors will not see the banner again until it is cleared.
